
Requirements:
NodeJs to be installed in the system.

How to install:
Extract the zip file.

1.Open the extracted folder with an ide of choice , used visual studio code.
2.Open the intergrated terminal of the ide.
3.install needed dependencies by running (npm install).
4. Start the project by running npm start .

If you access the  localhost on port 3000, http://localhost:3000/ ,in a web browser, you can observe the Index page.
